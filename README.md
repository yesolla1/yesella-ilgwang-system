# 📚 예설라 일광원 입학 상담 관리 시스템

초등 독서 학원 입학 상담 데이터를 디지털화하고 스마트한 시간표 관리를 제공하는 시스템입니다.

## ✨ 주요 기능

- 권한 기반 접근 제어 (관리자/직원)
- 학생 정보 CRM 관리
- OCR 수기 원서 디지털화
- 스마트 시간표 스케줄링
- 우선순위 자동 계산

## 🔐 로그인 정보

- 아이디: `admin`
- 비밀번호: `admin123`

⚠️ 최초 로그인 후 비밀번호를 변경하세요.

## 🛠️ 기술 스택

- Frontend: Streamlit
- Database: Supabase (PostgreSQL)
- Deployment: Streamlit Cloud
